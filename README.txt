# HW2
Build the project using build.xml file

Prerequisite:
Apache Ant
Java JDK

Run the file using command
ant build.xml


Run Java Project (#Main class name is TableLamp)

Ant build will generate Parth-Joshi-HW-2.jar in project directory.

Run jar file using command
java -jar Parth-Joshi-HW-2.jar
